HTML For Beginner

HTML is the language in which most websites are written. HTML is used to create pages and make them functional.

The code used to make them visually appealing is known as CSS and we shall focus on this in a later tutorial. For now, we will focus on teaching you how to build rather than design.

What you'll learn:
  • Learn to use Object Oriented Programming with classes!
  • Understand complex topics, like decorators.
Requirements:
  • Internet-connected computer (Windows, Ubuntu or MacOS).
  • High sense of self-study, high responsibility, persistence, not afraid of difficulties.
  • You don't need to know anything more, in the course I'll show you what you need to know.
